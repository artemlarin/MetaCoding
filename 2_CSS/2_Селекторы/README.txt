где стили описаны в отдельном файле:

1. Используя селектор *, окрасьте всю страницу любым цветом на ваш выбор.

3. Присвойте идентификатор тексту последней новости, например, 'sport'.

4. Опишите стили для идентификатора 'sport': установите большой размер шрифта и сделайте текст курсивным.
