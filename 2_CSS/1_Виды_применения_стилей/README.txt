Инструкции по предоставлению домашнего задания для модуля CSS

Поскольку в CSS имеются три способа написания кода:

1. Внедрение CSS стилей непосредственно внутри обычного HTML файла.
2. Размещение всех стилей внутри блока <head> </head>
3. Использование ссылки на файл с CSS.

Самым рекомендуемым способом является третий метод, который позволяет вам 
отправить всего лишь два документа на проверку домашнего задания.

ЗАДАНИЕ

Представьте себе, что необходимо разработать веб-страницу для новостного портала. 
На этой веб-странице должны присутствовать следующие элементы:

1. Главный заголовок (Пример: "Сегодняшние Новости", "Новости из Алматы").
2. Заголовок статьи.
3. Текст статьи.
4. Ссылка, ведущая на полную статью.


Таким образом, есть три типа элементов, которые должны быть стилизованы в одном стиле 
(Заголовок статьи, текст и ссылка – все они должны иметь единый дизайн).


Ваше задание состоит из трех этапов:

1. Создание веб-страницы с применением индивидуальных стилей для каждого элемента. 
Пожалуйста, пришлите этот файл в формате .html.

2. Внедрение стилей для всех указанных элементов непосредственно внутри тегов в одном .html файле.

3. Создание файла style.css, в котором будут прописаны стили для всех трех указанных элементов 
(т.е. для тегов) в отдельном файле .css. Кроме того, отправьте два файла: .html и .css.

Пожалуйста, выполните задание согласно этим трем этапам и предоставьте соответствующие файлы для каждого этапа
