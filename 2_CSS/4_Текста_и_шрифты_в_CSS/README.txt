ЗАДАНИЕ

Исходя из файла из урока 5, вам потребуется выполнить следующие действия:

1. Примените шрифт Montserrat к обычному тексту, используя шрифты Google.

2. Назначьте шрифт Georgia для Главного заголовка. Для этого используйте шрифт Georgia из библиотеки Visual Studio Code.

3. Примените шрифт Roboto к заголовкам новостей, воспользовавшись шрифтами Google.

4. Осуществите выравнивание по центру для заголовков третьего уровня.

5. Вставьте подсветку фона текста любого цвета в случайное место в тексте.
