ЗАДАНИЕ

Для кода, предоставленного в предыдущем домашнем задании, примените параметры, описанные в уроке, к каждой ссылке:


1. Измените цвет для новой не посещенной ссылки.


2. Задайте новый цвет для посещенной ссылки.


3. При наведении курсора мыши на ссылку, укажите цвет ссылки и измените цвет фона.


4. Определите цвет фона и цвет ссылки для активной ссылки.
