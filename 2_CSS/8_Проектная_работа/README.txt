ЗАДАНИЕ

Требования для выполнения проекта:

1. Примените стили к элементам страницы в отдельном CSS файле.

2. Обязательно обеспечьте адаптивность новостного сайта в точно таком же виде, как было продемонстрировано. 
Сужение должно касаться только ширины объектов, 
при этом все элементы должны оставаться выровненными по центру и не должны перекрываться.

3. При наведении курсора на новостные блоки (включая фото, текст и div-элементы) 
а также на навигационные элементы на странице банка, курсор должен меняться на указатель. 
Для этого используйте псевдокласс hover и свойство cursor.

4. Реализуйте верхнее меню таким образом, чтобы оно было кликабельным, выровненным по центру и адаптивным. 
Только надписи должны быть кликабельными, не применяйте кликабельность к div-элементам.

5. Используйте шрифт Roboto.

6. На странице навигации банка, сделайте кнопки адаптивными и обязательно добавьте стрелки. 
Вы можете добавить стрелки с помощью...

7. Текст «10% Kaspi Магазин» должен быть размещен поверх изображения.

8. Вставьте карту банка в конец страницы с использованием указанного сервиса и тега.
