Введение в HTML

1. Создайте файл с именем "helloworld.html".

2. Внутри файла "helloworld.html" добавьте следующую строку в раздел <head>: 
<meta charset="utf-8"- Эта строка указывает браузеру, какую кодировку символов 
следует использовать для отображения содержимого страницы.

3. Добавьте заголовок вашей страницы внутри раздела <head>:   
<title>Моя первая страница</title> - Этот заголовок будет отображаться в верхней части вкладки браузера.

4. Внутри тега <body> добавьте текстовый блок:

<p>Hello World! This is my first web-page!</p>- Этот текст будет отображаться на вашей веб-странице.

5. Сохраните файл "helloworld.html".

6. Прикрепите и отправьте файл "helloworld.html" к этому домашнему заданию.
