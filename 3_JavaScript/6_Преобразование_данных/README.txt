ЗАДАНИЕ

1. Создайте новый HTML файл (например, "index.html") и в этом файле подключите внешний файл со скриптом JavaScript
(назовем его "script.js"). Можно из предыдущих уроков скопировать код.

2. Внутри "script.js" выполните следующие шаги:

   а) Объявите переменные разных типов данных: строку, число, булевое значение и массив.

   б) Используя функции преобразования данных, измените типы переменных:

       	Преобразуйте число в строку.

       	Преобразуйте строку в число.

	Преобразуйте булевое значение в число.

	Преобразуйте число в булевое значение.

3. Выполните различные операции с преобразованными данными:

	Сложите строку и число.

	Умножьте число на 3.

	Сравните булевое значение с числом и сохраните результат в переменной.

	Используя операторы сравнения, сравните преобразованное число с оригинальным числом и выведите результат на экран.

4. С помощью console.log() выведите результаты операций и преобразований данных в консоль браузера.

5. Откройте HTML файл в вашем веб-браузере, откройте консоль разработчика и проверьте вывод ваших операций и преобразований.
