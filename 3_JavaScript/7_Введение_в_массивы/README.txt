ЗАДАНИЕ

1. Создайте новый HTML файл (например, "index.html") и в этом файле подключите внешний файл со скриптом JavaScript
   (назовем его "script.js").

2. Внутри "script.js" выполните следующие шаги:

    a. Создайте массив numbers и заполните его числами от 1 до 10.

    b. Выведите в консоль значение третьего элемента массива.

    c. Измените значение пятого элемента массива на 15.

    d. Создайте многомерный массив matrix, который будет содержать 3 вложенных массива, 
       каждый из которых будет содержать числа от 1 до 3.

    e. Выведите в консоль второй элемент первого вложенного массива из matrix.

    f. Измените значение третьего элемента второго вложенного массива из matrix на 10.

    g. Выведите в консоль всю структуру массива matrix и убедитесь, что изменения внесены.

3. Откройте HTML файл в вашем веб-браузере, откройте консоль разработчика и проверьте вывод операций и изменений,
   выполненных вами в скрипте.
