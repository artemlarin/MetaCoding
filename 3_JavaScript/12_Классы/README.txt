ЗАДАНИЕ

1. Создайте новый HTML файл (например, "index.html") и подключите внешний файл со скриптом 
   JavaScript (назовем его "script.js").
2. Внутри "script.js" выполните следующие шаги:

    a. Определите класс "Студент" (Student) с тремя свойствами: Имя (name), Возраст (age) и Курс (course).
    b. Добавьте методы класса:
     	- `showInfo()`: Метод должен выводить информацию о студенте в формате 
	   "Имя: {name}, Возраст: {age}, Курс: {course}".
        - `increaseAge()`: Метод должен увеличивать возраст студента на 1.
3. Создайте два объекта студента на основе класса "Студент" с различными значениями свойств:

        - Первый студент с именем "Иван", возрастом 20 и курсом 3.
        - Второй студент с именем "Анна", возрастом 22 и курсом 4.
4. Вызовите методы объектов студентов для вывода информации о них и увеличения их возраста.
5. Откройте HTML файл в вашем веб-браузере, откройте консоль разработчика и проверьте 
   вывод информации о студентах и их увеличенных возрастах.
6. Сохраните файлы и отправьте мне обратно "index.html" и "script.js" с вашим выполненным заданием.
