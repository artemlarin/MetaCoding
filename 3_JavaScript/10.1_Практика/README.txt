ЗАДАНИЕ

1. Создайте новый HTML файл (например, "index.html") и в этом файле подключите внешний файл со скриптом
   JavaScript (назовем его "script.js").
2. Внутри "script.js" выполните следующие шаги:
  Задание 1: Средняя длина слов в массиве строк

	a. Создайте массив строк, например: `["apple", "banana", "cherry", "date", "elderberry"]`.
	b. Найдите среднюю длину слов в массиве. 
	Для этого посчитайте сумму длин всех слов и поделите на количество слов.
        c. Выведите массив слов и среднюю длину слов в консоль с помощью `console.log()`.

  Задание 2: Наименьшее число в массиве

        a. Создайте массив чисел, например: `[34, 12, 45, 7, 23, 9]`.
        b. Определите наименьшее число в массиве. 
	   Для этого пройдитесь по массиву и найдите минимальное значение.
        c. Выведите массив чисел и наименьшее число в консоль с помощью `console.log()`.

  Задание 3: Сумма квадратов чисел в массиве

        a. Создайте массив чисел, например: `[2, 4, 6, 8, 10]`.
        b. Найдите сумму квадратов всех чисел в массиве. 
	Для этого возводите каждое число в квадрат и складывайте результаты.
        c. Выведите массив чисел и сумму квадратов в консоль с помощью `console.log()`.

3. Откройте HTML файл в вашем веб-браузере, откройте консоль разработчика и 
   проверьте вывод результатов для каждого из трех заданий.
4. Сохраните файлы и пришлите мне обратно "index.html" и "script.js" с вашим выполненным заданием.
