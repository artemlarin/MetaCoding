ЗАДАНИЕ

1. Создайте новый HTML файл (например, "functions.html") и в этом файле подключите внешний файл со скриптом 
   JavaScript (назовем его "functions.js").
2. Внутри "functions.js" выполните следующие шаги:

    a. Напишите обычную функцию `addNumbers`, которая принимает два числа в качестве аргументов и возвращает их сумму.
    b. Перепишите функцию `addNumbers` в виде стрелочной функции `addNumbersArrow`.
    c. Напишите обычную функцию `multiplyArrayElements`, которая принимает массив чисел и возвращает новый массив, 
       в котором каждый элемент умножен на 2.
    d. Перепишите функцию `multiplyArrayElements` в виде стрелочной функции `multiplyArrayElementsArrow`.
    e. Напишите обычную функцию `checkEvenOdd`, которая принимает число и возвращает строку "Четное" или "Нечетное" 
       в зависимости от четности числа.
    f. Перепишите функцию `checkEvenOdd` в виде стрелочной функции `checkEvenOddArrow`.

3. В файле "functions.js" создайте несколько вызовов для каждой из функций и используйте `console.log()`, 
   чтобы вывести результаты в консоль браузера.
4. Откройте HTML файл в вашем веб-браузере, откройте консоль разработчика и проверьте вывод результатов ваших функций.
5. Сохраните файлы и пришлите мне обратно "functions.html" и "functions.js" с вашим выполненным заданием.
