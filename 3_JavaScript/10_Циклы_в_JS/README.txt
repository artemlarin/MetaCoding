ЗАДАНИЕ

1. Создайте новый HTML файл (например, "index.html") и подключите внешний файл со скриптом 
   JavaScript (назовем его "script.js").
2. Внутри "script.js" выполните следующие задания:

   Задание 1: Вычисление произведения чисел с использованием цикла for: 
   Напишите программу, которая будет вычислять и выводить произведение всех чисел от 1 до 10.

   Задание 2: Умножение элементов массива на 2: Создайте массив чисел и напишите программу, 
   которая умножит каждый элемент этого массива на 2 и выведет исходный массив и результат умножения.

   Задание 3: Вывод последовательности чисел с использованием цикла while: 
   Напишите программу, которая будет выводить следующую последовательность чисел: 1, 2, 4, 8, 16, 32, 64, 128, ... 
   Каждое следующее число в последовательности получается умножением предыдущего числа на 2. 

3. С помощью `console.log()` выведите результаты выполнения каждого задания на консоль.
4. Откройте HTML файл в вашем веб-браузере, откройте консоль разработчика и 
   проверьте вывод результатов выполнения каждого задания.
