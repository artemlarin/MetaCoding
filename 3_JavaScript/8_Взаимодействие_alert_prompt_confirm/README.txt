ЗАДАНИЕ

1. Создайте новый HTML файл (назовите его, например, "javascript_exercises.html").
2. Внутри HTML файла создайте раздел `<script>`.
3. Внутри раздела `<script>` выполните следующие задачи:
   Задание 1: Возраст через 10 лет
   - Запросите у пользователя его возраст с помощью `prompt`.
   - Вычислите и выведите с использованием `alert`, сколько лет пользователю будет через 10 лет.
   Задание 2: Сумма двух чисел
   - Запросите у пользователя два числа с помощью `prompt`.
   - Вычислите и выведите с использованием `alert` сумму этих двух чисел.
   Задание 3: Приветствие с полным именем
   - Запросите у пользователя его имя и фамилию с помощью `prompt`.
   - Объедините имя и фамилию и выведите приветственное сообщение с использованием `alert`.
4. После написания кода для каждого задания, откройте HTML файл в вашем веб-браузере, 
следуйте инструкциям в окнах `prompt` и проверьте результаты, которые будут выводиться с помощью `alert`.

Это задание поможет вам попрактиковаться в использовании JavaScript для взаимодействия с пользователем
и выполнения различных вычислений в одном файле.
