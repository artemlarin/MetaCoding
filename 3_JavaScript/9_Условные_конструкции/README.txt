1. Создайте новый HTML файл (назовите его, например, "javascript_exercises.html").

2. Внутри HTML файла создайте раздел `<script>` и подключите внешний файл со скриптом JavaScript (назовем его "main.js") с помощью атрибута `src`.

3. Внутри "main.js" напишите программу, выполняющую следующие задачи:

Задание 1: Проверка возраста
   - Запросите у пользователя его возраст с помощью `prompt`.
   - Проверьте, является ли пользователь совершеннолетним (возраст 18 и старше).
   - Если пользователь совершеннолетний, выведите сообщение "Добро пожаловать!", если нет, выведите сообщение      "Извините, доступ запрещен!" с помощью `alert`.

Задание 2: Проверка числа
   - Запросите у пользователя число с помощью `prompt`.
   - Проверьте, является ли это число положительным, отрицательным или нулем.
   - Выведите соответствующее сообщение с помощью `alert`, например, "Число положительное", "Число отрицательное"      или "Число равно нулю".

Задание 3: Определение времени года
   - Запросите у пользователя текущий месяц (в виде числа от 1 до 12) с помощью `prompt`.
   - Проверьте, является ли введенное значение числом месяца весной, летом, осенью или зимой.
   - Выведите соответствующее сообщение с помощью `alert`, например, "Сейчас весна!", "Сейчас лето!", "Сейчас      осень!" или "Сейчас зима!".

Задание 4: Сравнение чисел
   - Запросите у пользователя два числа с помощью `prompt`.
   - Сравните эти числа и выведите сообщение с использованием `alert`, указывающее, какое число больше или если они      равны.

   Задание 5: День недели
   - Запросите у пользователя день недели (в виде числа от 1 до 7, где 1 - понедельник, 2 - вторник и т.д.) с      помощью `prompt`.
   - Выведите сообщение с использованием `alert`, указывающее, выходной ли это день (суббота или воскресенье) или      будний день.

4. После написания кода для каждого задания, откройте HTML файл "javascript_exercises.html" в вашем веб-браузере,    следуйте инструкциям в окнах `prompt` и проверьте результаты, которые будут выводиться с помощью `alert`.

Это задание поможет вам попрактиковаться в использовании JavaScript для взаимодействия с пользователем и выполнения различных вычислений в одном файле.
